# JustMC-code-challenger
Модуль на Minecraft сервере [JustMC](#JustMC) на задания по коду, который возможно обучит кого-то основам при должном количестве заданий. [Как предложить задание?](#как-предложить-задание)

## Как предложить задание?
Для того чтобы предложить своё задание, нужно создать [pull request](https://github.com/JustKorzhik/JustMC-code-challenger/pulls). В запросе укажите ID вашего задания. Задание должно представлять собой модуль с двумя функциями: `#test.(id)`, `#space.(id)`. Для функций есть требования:
- `#test`: требуется вызов `#space` на каждую проверку выполнение задания. Желательно добавлять несколько разных вызовов функции при разных обстоятельствах.
- `#space`: требуется игнорируемая отправка сообщения, которое содержит информацию для выполнения задания, в котором объясняется какие аргументы нужно создать в функции, что возвращать и сделать.
